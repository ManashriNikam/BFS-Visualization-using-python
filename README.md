This project demonstrates Breadth-First Search (BFS) through an interactive and visual approach using Python. The goal is to help learners clearly understand how BFS explores nodes level by level in a graph or grid.

Instead of only printing traversal order, this project visually shows each step of the BFS algorithm, making it easier to grasp concepts like queue operations, visited nodes, and traversal flow.
